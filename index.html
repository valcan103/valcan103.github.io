Ninestars
  • html
  • css(sass)
  • js(jquery)
html과 css 실력을 기르기 위해서 웹 무료 템플릿 중
ninestars의 메인 화면을 따라 만드는 연습을 했습니다.
색 코드와 이미지 등 디자인 부분을 제외하고 최대한 비슷하게 스스로 만들어 보았으며,
화면 크기에 따른 사이트의 변화 또한 비슷하게 만들어 보았습니다.
또한, jquery를 이용하여 몇몇 기능들을 만들어 보며 연습해 볼 수 있었습니다.

기억에 남는 내용
jquery를 이용하여 필터 기능을 새롭게 배울 수 있었습니다.
먼저 직접 만들어 보기 위해 해당하는 html요소에 데이터 속성을 부여하여
표시할 내용과 일치하는 경우 보여지게 만들어서 대략적으로 기능을 완성하였고,
조금 더 깔끔하게 만들기 위해 검색을 통해 filter 기능을 배워
데이터 속성 대신 class를 이용하여 코드를 간략하게 만들 수 있었습니다.

아쉬운 내용
먼저 드롭다운 메뉴를 만드는데 시간이 생각보다 오래 걸렸던 점이 아쉬웠고,
filter 기능에서 특정 필터에서 다른 특정 필터로 바꿀때는 버벅 거림이 없었지만,
전체에서 특정 필터로, 특정 필터에서 전체로 바꾸는 경우 약간 어색하게 움직이는 점이 아쉬웠고,
해당하는 내용에 따라 네비게이션 바의 색이 들어오게 만들 때 직접적으로 스크롤량을 체크하여 구현하긴 했지만,
코드가 불필요하게 길어지는 느낌이 들어 아쉬웠습니다.

Illdy
  • html
  • css(sass)
  • js(jquery)
html과 css 실력을 기르기 위해서 웹 무료 템플릿 중 2번째로
illdy의 메인 화면을 따라 만드는 연습을 했습니다.
마찬가지로 색 코드와 이미지 등 디자인 부분을 제외하고 최대한 비슷하게 스스로 만들어 보았으며,
화면 크기에 따른 사이트의 변화 또한 비슷하게 만들어 보았습니다.
또, jquery를 이용하여 몇몇 기능들을 연습하고, 새롭게 배워 볼 수 있었습니다.

기억에 남는 내용
background-attatchment를 fixed로 설정하여 스크롤 할 때 새로운 느낌을 받았고,
빗금의 배경 이미지가 이미지 그 자체가 아닌 타일 하나를 반복시켜
빗금 이미지를 만들 수 있다는 것을 활용할 수 있게 되었고,
jquery의 animate 를 이용하여 정해진 목표 수치까지 애니메이션으로 숫자를 증가시킬 수 있는 방법을 새롭게 배웠습니다.

아쉬운 내용
미디어 쿼리를 이용할 때 배경 이미지를 자연스럽게 늘리거나 줄이는 부분에서
아직 css를 이용하여 background-image에 효과를 주는데 미숙함을 느꼈고,
정해진 수치까지 숫자가 증가하는 애니메이션을 먼저 직접 만들어 보려고 했지만,
생각했던 결과와는 다른 결과가 나와서 검색을 통해 해결했던 점이 아쉬웠습니다.

jquery 연습, 복습을 하기 위해 강의를 통해 배웠던 기능이나,
만들어 보고 싶었던 기능을 직접 만들어보는 연습을 했습니다.
먼저 일정량 스크롤 하면 네비게이션 바가 고정되게 만들어보았고,
강의에서 배운 캐러셀 갤러리 복습, 호버 이펙트 복습,

기억에 남는 내용
강의에서 배웠던 slick.js를 이용하여 활용하는 법을 복습했고,
jquery 공부를 간단하게 한 뒤,
관심이 있었던 아코디언 메뉴를 스스로 만들 수 있어서 뿌듯했고,
모달 창도 기능은 제대로 구현한 것 같아서 만족스러웠습니다.

아쉬운 내용
아직 자바 스크립트를 이용한 기능들을 많이 알지 못해서 많이 만들어 보지 못한게 아쉬웠고,
페이지를 열었을 때 기능을 처음으로 작동시키면 버벅거림이 있다는 점이 아쉬웠습니다.

React연습
  • html
  • css
  • js
  • React
html과 css를 자세하게 공부 하기 전,
리액트를 이용하여 유튜브를 비슷하게 만들어 보는 강의를 본 후
강의에서 배운 내용들을 바탕으로 많은 부분을 참고하여 게시판을 만들어 보는 연습을 했습니다.
아직 많은 부분을 이해하진 못했지만, 리액트를 경험해 볼 수 있는 기회였습니다.

(회원가입, 로그인, 로그아웃, 정보수정, 관리자 기능, 게시물 관리(쓰기, 수정, 삭제, 공개 범위 설정),
내가 쓴 글 보기, 게시물 즐겨찾기, 댓글 관리(쓰기, 수정, 삭제) 등의 기능을 연습했습니다.)

기억에 남는 내용
배웠던 내용 중 비공개 게시물은 다른 사람들에게는 보이지 않게,
관리자에게는 수정 및 삭제까지 관리 할 수 있도록 추가적으로 만들었던 점과,
강의에서 댓글 삭제 기능을 만들고 확인해 봤을 때,
상위 댓글을 지우면 하위 댓글들이 게시판에서는 삭제된 것 처럼 보이지만,
실제 mongoDb에서는 삭제되지 않아서 직접 생각하여 기능을 완성시킨 점이 가장 기억에 남았습니다.

아쉬운 내용
css를 자세하게 배우기 전이라 만들고 싶었던 기능들에만 집중했어서 보이는 화면이 어색한 점과,
게시글을 원하는 만큼 표시하는 방법, 페이지를 나누는 방법 등 아직 구현해 보지 못했던 기능들이 있었고,
리액트 경험을 목적으로 했어서 아직 리액트를 능숙하게 사용하지 못하는 점이 아쉬웠습니다.

React_todo
  • html
  • css
  • js
  • React
리액트 연습을 하기 위해 todo list를 만들어보는 연습을 했습니다.
시계 페이지도 만들어 보면서 페이지를 옮기는 방법을 복습했으며,
todo list를 만들어보는 과정에서 js6 문법도 어느정도 복습을 했습니다.
(현새 시간 보기, 할 일 추가, 삭제, 할 일 모두 지우기, 수정, 한 일 체크 등의 기능을 만들었습니다)

기억에 남는 내용
할 일들을 저장하는데 localStorage를 이용하여 저장하는 방법을 새롭게 공부했으며,
시간을 한국 시간에 맞춰서 받아오는 방법을 알게 되었고, 날짜를 받아오는 법을 복습했습니다.
이번 복습을 통해 리액트의 기초적인 부분은 어느정도 익숙해 진 것 같다고 느꼈습니다.

아쉬운 내용
todo 부분을 처음부터 생각을 해서 부분 부분 나눠서 해야 했는지,
한번에 같은 부분에 만들었더니 생각보다 너무 길어진 것 같아 아쉬웠고,
인터넷 도움 없이 처음부터 다 만들기에는 아직 새롭게 배워야 할 내용이 많다는 생각이 들었습니다.
